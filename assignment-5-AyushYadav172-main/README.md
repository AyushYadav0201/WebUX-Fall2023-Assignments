# Word Count Function

This JavaScript function, `countWords`, is designed to count the occurrences of each word in a given input sentence and return the results as a key-value pair in an object. The key represents the word, and the value represents the count of that word within the sentence.

# Filter and Reverse Utility

This utility function, `filterAndReverse`, is designed to filter a numeric array based on a given callback function and return a shallow copy of the reversed array with the filtered elements. This allows you to efficiently filter and reverse arrays based on specific criteria.

# HTMLNode Class

The `HTMLNode` class represents HTML elements and provides a method for selecting specific elements within a Document Object Model (DOM) tree.
The `HTMLNode` class is a JavaScript class designed to represent HTML elements within a DOM tree. It allows you to create and manipulate a DOM-like structure using JavaScript objects. The class provides a `search` method that enables you to select and retrieve specific elements based on tag names, CSS classes, and IDs.
