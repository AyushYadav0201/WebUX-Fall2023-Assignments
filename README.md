# INFO 6150 Assignments

**Web Design and User Experience Engineering – Fall 2023**

This repository contains **Assignments 3 through 9** and related lab work for **INFO 6150: Web Design and User Experience Engineering**, taught by **Amuthan Arulraj** at Northeastern University. The assignments focus on modern web development, including HTML, CSS, JavaScript, React, Redux, REST APIs, and other frontend and backend technologies.

---

## **Course Information**

* **Course Title:** Web Design and User Experience Engineering
* **Course Number:** INFO 6150
* **Term & Year:** Fall 2023
* **Credit Hours:** 4
* **Instructor:** Amuthan Arulraj ([aarulraj@northeastern.edu](mailto:aarulraj@northeastern.edu))

**Course Description:**
This course introduces the fundamentals of modern web development, design patterns, and frameworks. It focuses on developing complex web applications using JavaScript-based technologies along with best practices and design principles. Topics include HTML, CSS, JavaScript, ES6, React.js, Redux, React Router, Next.js, REST APIs, Node.js, MongoDB, TypeScript, and UI testing with Jest and Playwright.
